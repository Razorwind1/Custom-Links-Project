LinkTailor

A virtual wardrobe for your links...

Like a wardrobe full of clothes, LinkTailor is a virtual space where you can place applications, files, and websites – treating them as garments that you can style, tag, and organize into outfits for any occasion. By saving layouts of links within LinkTailor, you can easily access them all in one place or even open them all with a single click.


You might make a layout for...

Communications
Gmail · Outlook · Facebook · GroupMe · Instagram · Discord · Slack · Skype
Projects
Project documents · diagrams · your project portal · websites for research · a dropbox folder ·  a OneNote page
File System
Organize links to your file system! Save files and folders of: pictures, screenshots, artwork, resumes, essays, personal projects, records, etc.
Resource Sharing
Create a layout of websites to share information with others (eg: best restaurants, poverty assistance resources, best image editing websites, or a spectrum of political media).
Teaching Tutorial
Create a tutorial that moves the user from link to link, teaching a new concept.
Gaming
Display links to your favorite games in a beautiful special layout. Add other gaming resources as well (eg: Discord, Twitch, strategy websites, music, and folders with game saves).
"When I'm bored"
Fill a layout with activities you'd like to do when you're not sure what you want to do.
Podcasts
Save links to the podcasts you love or want to listen to and organize them based on genre or by your preference. You could share this layout with friends to share your podcasts.
"Things I love"
Create a layout to organize some of your favorite websites and resources.
Psychology 101
For any class you're in, you can make a layout to link all the important documents and websites you want to return to.

Features

  • Layouts: Arrange links spatially within a drag-and-drop grid system.
    • Grid size is adjustable
    • All links in a layout can be opened together.
  • Tags: Links can be given custom tags.
  • Styles: Links can be custom styled and given custom icons.
    • Icons are automatically fetched from the file system and websites.
  • Windows Context Menu Integration: In Windows, files can be added to LinkTailor through the right-click menu (if the option is not deselected during install).
  • Undo: Use Ctrl+Z to undo your last action.
